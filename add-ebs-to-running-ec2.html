稼働中のEC2インスタンスにディスク(EBS)を追加する

  • 2015.06.05
  • AWS
  • masutaka

こんにちは。増田です。

このブログの記事の作成から公開まで、エンジニア以外の方向けに社内でデモすることになりました。
これはその記事です。

ディスクを追加

  1. EC2の管理画面からELASTIC BLOCK STORE > Volumesに進み、希望サイズのボリュームを作る
  2. Available になったら、マウントしたいEC2インスタンスにAttachする

  3. EC2インスタンスにsshログインし、ファイルシステムを作成、マウントする

    $ ssh test1.example.jp
$ ls -alF /dev/xvdk
brw-rw---- 1 root disk 202, 160  4月  3 15:29 2015 /dev/xvdk

$ sudo mkfs.ext4 /dev/xvdk

$ sudo mkdir /hoge
$ sudo mount /dev/xvdk /hoge

  4. 60GBのボリュームがマウントされた。

    $ df /hoge
Filesystem           1K-ブロック    使用   使用可 使用% マウント位置
/dev/xvdk             61927420    184136  58597556   1% /hoge

他のEC2インスタンスに繋ぎかえる

他のEC2インスタンスでマウントすることも可能

  1. unmount

    $ sudo umount /hoge

  2. EC2の管理画面からDetach、マウントしたいEC2インスタンスにAttachする

  3. マウントしたいEC2インスタンスにsshログインし、mountする

    $ ssh test2.example.jp
$ sudo mkdir /hoge
$ sudo mount /dev/xvdk /hoge
